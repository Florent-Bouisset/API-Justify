# API-Justify

Doc de l'API : https://documenter.getpostman.com/view/12880848/TVYKZGCY#c466f686-08d2-4ff1-b5c7-002720ddc777

### Installation

Prérequis: node et mongo installé sur la machine
```
  npm install
  npm run build
```  
  
 ### Lancement sur port 3000: 
 ```
 npm run start
 ```  

 ### Lancement sur port 5566: 
 ```
 npm run prod
 ```
 
 ### Tests jest:
 ```
 npm run test
 ```
 
 ### lint:
 ```
 npm run lint
 ```
